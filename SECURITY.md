# Security Policy

Please do not open a public issue for a suspected security vulnerability in the verifier, initializer, or installed skill package.

Use [GitHub private vulnerability reporting](https://github.com/scottfalconer/agent-ready-drupal-build-kit/security/advisories/new) and include the affected command or file, impact, reproduction steps, and any safe supporting evidence. Do not include credentials, private source-site content, or customer review packets.

Ordinary bugs, documentation problems, and feature proposals belong in the public issue tracker.
