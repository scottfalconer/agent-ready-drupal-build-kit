# Publication Checklist

Before sharing a kit package publicly:

- Remove local paths.
- Remove private notes.
- Remove real third-party source-site content unless explicit authorization exists.
- Remove generated source code unless publishing an implementation package.
- Remove disposable lab databases and exports.
- Remove screenshots and media assets from source sites.
- Use fictional examples or explicitly permissioned examples.
- Include concrete evidence requirements.
- Include license and contribution terms appropriate for the publication venue.
- Run a text scan for local usernames, private domains, credentials, and accidental source content.
